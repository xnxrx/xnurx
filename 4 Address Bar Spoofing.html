🎭 POPUP ADDRESS BAR SPOOFING

New Window / New Tab • URL Google asli • Kontol dari Attacker
✅ TEKNIK YANG DIGUNAKAN:
• window.open() ke URL Google asli (google.com/csi, accounts.google.com, etc.)
• setTimeout + document.write() untuk menimpa konten dengan halaman phishing "Google says..."
• history.replaceState() untuk membersihkan jejak (optional)
• Hasil: Address bar menunjukkan URL Google tapi kontol berasal dari attacker!

🎯 Interpretasi Hasil

  • Address Bar = Google.com, Konten = Phishing → 🎉 SPOOFING BERHASIL! Browser rentan terhadap address bar spoofing.
  • Popup terbuka meskipun popup blocker aktif → Bypass berhasil.
  • Address Bar berubah menjadi about:blank atau URL lain → Spoiling gagal, browser sudah patched.
  • Popup tidak terbuka sama sekali → Popup blocker aktif, izinkan popup untuk pengujian.

📝 Cara Penggunaan

1. Simpan file sebagai popup-spoof.html
2. Buka di browser target (Safari, Chrome, Firefox, Edge)
3. Izinkan pop-up jika diminta
4. Klik tombol dan amati: Address bar di popup baru menunjukkan URL Google tapi kontol berasal dari attacker!
5. Jika berhasil, screenshot sebagai bukti temuan.
