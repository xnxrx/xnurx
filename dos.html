⚡ VECTOR 16 - FULLSCREEN NOTIFICATION OVERLAY ⚡

Metode Menutupi Notifikasi Fullscreen Browser (UI Redressing)
🔬 ANALISIS TEKNIS:

Notifikasi fullscreen browser (contoh: "google.com wants to enter full screen") adalah security feature yang tidak bisa ditutup/dihilangkan oleh JavaScript.

Namun, notifikasi tersebut bisa DITUTUPI SECARA VISUAL menggunakan berbagai metode overlay:
✅ Z-Index Max - Element dengan z-index lebih tinggi menimpa notifikasi
✅ PiP Window - Picture-in-Picture window bisa muncul di atas notifikasi
✅ Video Element - Native video PiP memiliki z-index tertinggi
✅ Alert/Prompt - Modal dialog browser juga muncul di atas
✅ Drag & Drop - Native drag image memiliki layer sendiri
✅ Fullscreen Element - Element fullscreen bisa menimpa notifikasi
⚠️ Catatan: Ini hanya MENUTUPI secara visual, bukan menghapus notifikasi

📦 METHOD 1: Z-Index Max Overlay

Membuat div dengan z-index: 2147483647 (nilai maksimal integer) untuk menutupi notifikasi.

🖼️ METHOD 2: Picture-in-Picture Native PiP

Native PiP window memiliki layer tertinggi di browser.

🎬 METHOD 3: Video PiP API Native

Menggunakan requestPictureInPicture() API dari video element.

🖱️ METHOD 4: Drag Image Native

Native drag image memiliki layer sendiri di atas browser UI.

💬 METHOD 5: Browser Modal Native

alert(), confirm(), prompt() memiliki z-index tinggi.

🖥️ METHOD 6: Fullscreen Div Overlay

Membuat element fullscreen yang menimpa UI browser.
